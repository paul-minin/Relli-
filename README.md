# Relli-

Mini Top-Down Car Game

Kurz: Ein einfaches Browser-Spiel (HTML + Canvas + JS) mit Top-Down-Ansicht. Steuerung per Pfeiltasten. Es gibt Hindernisse und sichtbare Ränder (Wände).

So starten:
- Öffne `index.html` im Browser (oder nutze Live Server in VS Code)

Steuerung:
- Pfeiltasten oder WASD: Vorwärts / Rückwärts / Links / Rechts
- Leertaste: Drift (halten)
- R: Reset (Startposition)

Dateien:
- `index.html` – Grundgerüst
- `style.css` – Einfache Styles
- `main.js` – Spiel-Logik
